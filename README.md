###  Featured Project

#### [🚦 look_Around — Bengaluru Parking Intelligence](https://github.com/Ashish62636/look_Around-Parking_Intelligence)
An AI-driven spatial parking-violation intelligence system built during a ~48h hackathon. It ingests **298,450 records** and serves real-time prediction overlays:
- **Spatial Clustering (H3 & DBSCAN):** Density-based clustering of chronic violation areas.
- **Predictive Congestion Score (LightGBM):** Machine learning model predicting localized congestion risk with 98.5% confidence.
- **Enforcement Priority Queue:** WebGL dashboard dispatch interface sorting spatial hotspots into real-time dispatch tiers.

*Stack:* React, TypeScript, Tailwind CSS, deck.gl, FastAPI, DuckDB, LightGBM, Git, Docker.
